# YAHT Privacy Policy

Last updated: July 3, 2026

YAHT Habit Tracker is a local-first habit tracking app. The app is designed to keep habit data on your device unless you choose to export or share it.

## Data Stored on Your Device

YAHT stores the following app data locally:

- Habit names, icons, schedules, reminder settings, completion history, timer state, and statistics derived from that data.
- App settings such as theme, first day of week, and timed habit behavior.
- Runtime reminder ledgers used to schedule, deduplicate, snooze, and clear local notifications.
- Privacy-safe diagnostic events used for troubleshooting notification and app behavior.

This data is stored in the app sandbox using local device storage. YAHT does not require an account and does not send this data to a YAHT server.

## Notifications

If you enable reminders or timers, YAHT schedules local notifications on your device. Notification titles may include habit names so the reminder is useful to you. Notification quick actions can mark a habit done or snooze a reminder locally.

On Android, YAHT uses exact alarms for user-scheduled reminders and timers so they can fire at the time you chose. If exact alarm access is disabled, reminder and timer notification reliability can be reduced.

## Import, Export, Sharing, and Diagnostics

YAHT includes user-initiated import and export features:

- Exporting habit data creates a JSON backup file that may include your habit names, schedules, reminders, completion history, and timer data.
- Importing habit data reads the JSON file you select.
- Exporting diagnostics creates a JSON report with app version, platform, locale, timezone, theme mode, and privacy-safe diagnostic events. Diagnostics intentionally exclude habit titles, raw import/export files, notification body text, arbitrary objects, and stack traces.

When you share or save an exported file, the destination app or storage provider you choose may process that file under its own privacy policy.

## Data Collection and Sharing

YAHT does not collect user data to developer-operated servers. YAHT does not sell user data, use advertising SDKs, or track users across apps or websites.

Opening external links, such as the source code or support page, leaves YAHT and opens the selected website or browser. Those external services are governed by their own privacy policies.

## Data Deletion

You can delete app data from inside YAHT by using the reset data action in Settings. You can also delete YAHT app data by uninstalling the app or clearing app storage through your operating system.

Exported backup or diagnostic files are stored wherever you chose to save or share them. Delete those files from the destination location if you no longer want to keep them.

## Children

YAHT is not directed to children and does not knowingly collect personal data from children.

## Changes

This policy may be updated when YAHT changes how it stores, exports, or processes data. Material changes should be reflected in the app store privacy declarations before release.
